Remarks
This implementation of Min() is specialized for a CondensedCollection and tries to take advantage of the CondensedCollection's knowledge of unique values in order to improve performance. If you would rather use the normal LINQ extension method then cast the CondensedCollection to an IListT to use the MaxTSource, TResult(IEnumerableTSource, FuncTSource, TResult) implementation.
