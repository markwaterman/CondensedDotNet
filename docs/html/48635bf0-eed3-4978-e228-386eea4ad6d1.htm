Condensed.Linq
The Condensed.Linq namespace provides classes that support queries against Condensed collections using Language-Integrated Query (LINQ). The CondensedCollectionExtensions class contains extension methods that can be used in conjunction with the CondendesedCollection class to improve performance of several LINQ operations.
