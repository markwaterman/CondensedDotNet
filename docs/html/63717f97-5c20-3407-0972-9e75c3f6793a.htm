Computes the maximum value of a CondensedCollection of nullable float values that are obtained by invoking a transform function on each element of the input sequence. Optimized for a CondensedCollection.

Remarks
This implementation of Max() is specialized for a CondensedCollection and tries to take advantage of the CondensedCollection's knowledge of unique values in order to improve performance. If you would rather use the normal LINQ extension method then cast the CondensedCollection to an IListT to use the MaxTSource(IEnumerableTSource, FuncTSource, NullableSingle) implementation.
